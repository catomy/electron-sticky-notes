# 应用图标

请在此目录放置应用图标文件：

- Windows: `icon.ico` (256x256 推荐)
- macOS: `icon.icns` (512x512 推荐)

您可以使用以下工具生成图标：

1. [在线图标生成器](https://icoconvert.com/)
2. [Image2icon](https://www.img2icnsapp.com/) (macOS)
3. [GIMP](https://www.gimp.org/) (免费开源)

临时方案：
- 如果没有图标文件，应用将使用系统默认图标
- 或从 electron-builder 示例项目下载示例图标使用
